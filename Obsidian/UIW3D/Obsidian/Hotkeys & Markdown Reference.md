## Text Formatting

**Bold**
- Hotkey: `Ctrl + B`
- Syntax: `**Bold**` → **Bold**

**Italic**
- Hotkey: `Ctrl + I`
- Syntax: `*Italic*` → *Italic*

**Strikethrough**
- Syntax: `~~Strikethrough~~` → ~~Strikethrough~~

**Highlight**
- Syntax: `==Highlight==` → ==Highlight==

**Inline Code**
- Syntax: `` `code` `` → `code`

---

## Headings

**Heading 1**
- Syntax: `#`

**Heading 2**
- Syntax: `##`

**Heading 3**
- Syntax: `###`

---

## Lists

**Bulleted List**
- Syntax: `-` or `*`
- Bullet item
- Bullet item

**Numbered List**
- Syntax: `1.` `2.` `3.`
1. First item
2. Second item
3. Third item

**Checklist**
- Syntax: `- [ ]`
- [ ] Unchecked item
- [x] Checked item

---

## Quotes & Dividers

**Quote Block**
- Syntax: `>`
> This is a quote block

**Divider**
- Syntax: `---`

---

## Code

**Inline Code**
- Syntax: Wrap with single backticks
- Example: `` `code` `` → `code`

**Code Block**
- Syntax: Wrap with triple backticks
```python
# This is a code block
def hello_world():
    print("Hello, World!")
```

---

## Links & Embeds

**Link to Internal Note**
- Syntax: `[[Note Name]]`
- Example: [[What is PowerShell]]

**Link with Custom Text**
- Syntax: `[[Note Name|Display Text]]`
- Example: [[What is PowerShell|PowerShell Guide]]

**Embed Entire Note**
- Syntax: `![[Note Name]]`

**External Link**
- Syntax: `[Link Text](URL)`
- Example: [Obsidian Documentation](https://obsidian.md)

**Embed External Image**
- Syntax: `![Alt Text](URL)`
- Example: `![Logo](https://example.com/logo.svg)`
