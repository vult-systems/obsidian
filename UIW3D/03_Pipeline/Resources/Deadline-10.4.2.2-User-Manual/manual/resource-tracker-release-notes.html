Upgrading Resource Tracker¶

Warning

Starting from Deadline 10.0.10, Resource Tracker versioning was decoupled from Deadline versioning. Deadline Infrastructure will now always automatically use the latest version of Resource Tracker, and there is no need for manual upgrade.

More about Resource Tracker Versioning in: Resource Tracker Version

Resource Tracker 11.6.1¶

Complete Release Notes¶

Warning

The AWS Lambda service is deprecating the Python 3.7 runtime on November 27, 2023. For more information, see https://docs.aws.amazon.com/lambda/latest/dg/lambda-runtimes.html. Customers that have Resource Tracker deployments earlier than version 11.6.0 may have received notifications about this deprecation for Resource Tracker Lambda functions.

Resource Tracker will automatically upgrade when any of the following happens:

  • A new AWS Portal infrastructure is created

  • A new AWS Portal fleet request is made

  • Spot Event Plugin is enabled

Customers with the Resource Tracker auto upgrade feature enabled require no further action. This is the default behavior. For customers with this setting disabled, please enable the setting to allow the Resource Tracker to automatically upgrade to the latest version.

  • All Resource Tracker Lambdas have been updated to the Python 3.10 runtime.

Resource Tracker 11.5.2¶

Complete Release Notes¶

  • All Resource Tracker Lambdas have been updated to the Python 3.7 runtime.

Resource Tracker 11.4.0¶

Complete Release Notes¶

  • To reduce API throttling, the FleetTableCleaner Lambda now runs every 5 minutes instead of each time an instance is terminated.

Resource Tracker 11.3.1¶

Complete Release Notes¶

  • Fleet health is now calculated against the fleet’s fulfilled capacity instead of the target capacity.

Resource Tracker 11.3.0¶

Complete Release Notes¶

  • Unhealthy workers will be terminated instead of rebooting.

Resource Tracker 11.2.0¶

Complete Release Notes¶

  • Fixed throttling of EC2 API by limiting the concurrency of a Lambda function and updated the retry logic mode of the Botocore client to the newer ‘standard’ option.

Resource Tracker 11.1.0¶

Complete Release Notes¶

  • Added parameter to exclude tags for SQS queues in aws-us-gov.

Resource Tracker 11.0.0¶

Complete Release Notes¶

  • Added autoupgrade feature. Deadline Infrastructure will now always automatically use the latest version of Resource Tracker.
