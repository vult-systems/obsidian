Corona Standalone¶

Job Submission¶

You can submit Corona jobs from the Monitor.

Submission Options¶

The general Deadline options are explained in the Job Submission documentation. The Corona specific options are:

  • Corona Version: The version of Corona Standalone to use.

  • Corona Scene: The Corona scene that will be rendered. Must be a .scn file.

  • Output File Directory: The directory for the output to be saved to.

  • Output File Name: The prefix for the output file names. If not specified it defaults to “output”.

  • Save Alpha: If selected, the alpha output of the render will be saved.

  • Save Raw: If selected, the RAW output of the render will be saved.

  • Frame List: The list of frames to be rendered. Each frame will be rendered to a separate output file.

  • Single Frame Job: If selected, the job is a single frame job.

  • Configuration File(s): Add any configuration files for Corona here. Configuration files are processed in the order they are listed.

The Corona specific advanced options are:

  • Override maximum # of Passes: You can override the configuration file setting for the maximum number of passes here if this is enabled.

  • Override maximum Render Time: You can override the configuration file setting for the maximum render time here if this is enabled.

  • Override Threads: You can override the configuration file setting for the number of threads here if this is enabled.

Plugin Configuration¶

You can configure the Corona plugin settings from the Monitor. While in power user mode, select Tools -> Configure Plugins and select the Corona plugin from the list on the left.

Render Executables

  • Corona Standalone Render Executable: The path to the Corona Render executable file used for rendering. Enter alternative paths on separate lines.
