Shake¶

Job Submission¶

You can submit Shake jobs from the Monitor.

Submission Options¶

The general Deadline options are explained in the Job Submission documentation, and the Draft/Integration options are explained in the Draft and Integration documentation. The Shake specific options are:

  • Shake Script File: The Shake script file to be rendered.

  • CPUs: The number of CPUs to use during rendering.

  • Additional Arguments: Additional arguments to pass to the Shake command line renderer.

Plugin Configuration¶

You can configure the Shake plugin settings from the Deadline Monitor. While in power user mode, select Tools -> Plugins Configuration and select the Shake plugin from the list on the left.

Render Executables

  • Shake Executable: The path to the Shake executable file used for rendering. Enter alternative paths on separate lines.
