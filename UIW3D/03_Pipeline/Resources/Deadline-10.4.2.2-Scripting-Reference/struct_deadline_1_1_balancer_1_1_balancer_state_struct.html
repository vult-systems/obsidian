Detailed Description

A BalancerStateStruct holds a deep copy of all state information for consumption by a BalancerLogicPlugin. The BalancerStateStruct is strictly a value set, with no backward ties. The BalancerLogicPlugin is free to manipulate the struct for its own purposes if desired.
