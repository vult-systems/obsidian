Detailed Description

The managed wrapper for the python Cloud Plugins. The CloudPluginWrapper class uses a builder pattern to construct objects. In order for CloudPluginWrapper objects to function properly, they must set the PluginConfig, DataController, and CloudPluginName, then initialze the plugin secret retriever.
