static void Deadline.Scripting.SlaveUtils.SendRemoteCommand ( string  machineName,
string  command 
)
static

Send a remote command to the specified machine. Scripts that utilize this function are intended to be executed only from the Monitor. Use "SendRemoteCommandWithResults" to wait on a response from the Deadline Launcher.

The issued remote command is added to the Remote Commands panel in the Deadline Monitor.

Parameters
machineNameThe machine to send the command to.
commandThe command.

The available commands are StopLauncher, LaunchSlave [SLAVENAME], LaunchAllSlaves, LaunchNewSlave [SLAVENAME], RemoveSlave [SLAVENAME], LaunchSlaveDelay, StopSlave, RelaunchSlave, ForceStopSlave, ForceRelaunchSlave, OnLastTaskComplete [OPERATION] [MESSAGE], LaunchPulse, StopPulse, RestartPulse, RemotePulseCommand [OPERATION], LaunchWebService, StopWebService, RestartWebService, LaunchBalancer, StopBalancer, RestartBalancer, RemoteBalancerCommand [OPERATION], LaunchMonitor, StopMonitor, RestartMachine [MESSAGE], StartMachine, ShutdownMachine [MESSAGE], Execute [COMMAND], and ExecuteNoWait [COMMAND].

The [SLAVENAME] option is optional for the LaunchSlave command, and it is required for the LaunchNewSlave and RemoveSlave commands. For example:

SlaveUtils.SendRemoteCommand( "myMachine", "LaunchNewSlave newSlaveName" )

The [OPERATION] option is required for the OnLastTaskComplete command. Valid [OPERATION] options are Continue, StopSlave, RestartSlave, ShutdownMachine, RestartMachine, CheckForTasks, CancelTask, and ClearFailureCache. The [MESSAGE] option is not required but can be added for OnLastTaskComplete command. A valid [MESSAGE] is ': [content] :' where the [content] does NOT contain ':'. Here are two examples, one with a message and one without:

SlaveUtils.SendRemoteCommand( "myMachine", "OnLastTaskComplete StopSlave" )

SlaveUtils.SendRemoteCommand( "myMachine", "OnLastTaskComplete ShutdownMachine : myMessage Here :" )

The [OPERATION] option is required for the RemotePulseCommand command. Valid [OPERATION] options are HouseCleaning, PendingJobScan, RepositoryRepair, and PowerManagement. For example:

SlaveUtils.SendRemoteCommand( "myMachine", "RemotePulseCommand PendingJobScan" )

The [OPERATION] option is required for the RemoteBalancerCommand command. Valid [OPERATION] options are Balancing. For example:

SlaveUtils.SendRemoteCommand( "myMachine", "RemoteBalancerCommand Balancing" )

The [COMMAND] option is required for the Execute and ExecuteNoWait commands, and represents the command to run. For example:

SlaveUtils.SendRemoteCommand( "myMachine", "Execute cmd /C dir" )

The [MESSAGE] option is not required but can be added for the ShutdownMachine and RestartMachine command. A valid [MESSAGE] is '[content]' where the [content] does NOT contain ':'. For example:

SlaveUtils.SendRemoteCommand( "myMachine", "RestartMachine myMessage Here" )
