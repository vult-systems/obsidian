bool Deadline.Jobs.Task.Equals ( Task  other)

Checks whether or not this Task is the same as another, by comparing their internal properties. Note that this performs a deep compare of both objects, and does not just check for reference equality.

Parameters
otherThe Task that will be compared to the current instance.
Returns
True if the task objects are deemed to be the same, False otherwise.
